# Mini-CRM (CLI)

Небольшая консольная CRM на Python для ведения базы клиентов в **JSON-файле**. Позволяет добавлять клиентов, просматривать список, редактировать и удалять записи, выполнять поиск/фильтрацию/сортировку и смотреть простую статистику.


---

## Возможности

- **Добавление клиента** (валидация вводимых данных)
- **Просмотр всех клиентов**
- **Редактирование параметра клиента** по `id`
- **Удаление клиента** с подтверждением
- **Поиск** по любому полю клиента
- **Фильтры**:
  - по году основания ("раньше/позже" указанного года)
  - по дате добавления (интервал `YYYY-MM-DD`)
- **Сортировки**:
  - по году основания (возрастание/убывание)
  - по алфавиту (A-Z / Z-A)
- **Статистика**:
  - клиентов за последнюю неделю
  - отечественных/зарубежных клиентов (домашняя страна считается `Russia`)
  - клиентов со статусом `work`

---

## Технологии

- Python **3.10+** (используются аннотации вида `list[dict] | None`)
- Внешние зависимости **не требуются** (только стандартная библиотека)
